14种专业重命名规则

14 Professional Renaming Rules

Lightning Renamer 精心打造了14种功能强大的重命名规则，涵盖文件管理的方方面面。从基础的文本替换、插入、删除，到专业的序列化编号、大小写转换、扩展名修改；从实用的清理空白、去除字符、长度填充，到高级的正则表达式匹配、用户自定义输入、随机化处理以及全新加入的自定义脚本规则，每一种规则都经过精心设计，确保简单易用的同时提供强大的功能。

这些规则支持无限组合使用，并可通过直观的拖拽操作自由调整执行顺序。配合实时预览功能，您可以在应用前清晰看到每一步操作的结果，避免误操作。更强大的是，每条规则都支持"作用域"设置，让您能够精准控制规则应用的目标文件范围，实现真正意义上的灵活精准控制。

Lightning Renamer features 14 powerful renaming rules covering every aspect of file management. From basic text replacement, insertion, and deletion, to professional serialization numbering, case conversion, and extension modification; from practical whitespace cleaning, character stripping, and padding, to advanced regular expression matching, custom user input, randomization, and the newly added custom script rule. Each rule is carefully designed to ensure ease of use while delivering powerful functionality.

These rules support unlimited combinations and can be freely reordered through intuitive drag-and-drop operations. With real-time preview functionality, you can clearly see the results of each operation before applying changes, preventing errors.

手术刀般精准的"规则作用域"功能

Surgical Precision with "Rule Scope"

如同外科手术刀般精准，规则作用域让您能够将每条规则精确定位到指定文件，在复杂场景下实现极致的重命名控制。当您面对一个混杂着图片、视频和文档的文件夹时，您可以创建一条规则专门作用于所有.jpg文件，另一条规则只作用于文件名包含"报告"的文档，而第三条规则则应用到其余所有文件上。

作用域功能与文件筛选功能完美结合，允许您先用筛选器快速定位特定文件群组（如某个时间段创建的文件、特定大小范围的文件或特定扩展名的文件），然后为这些筛选后的文件单独设置规则，实现多层次、多条件的精准文件处理。

Like a surgeon's scalpel, Rule Scope enables you to precisely target each rule to specific files, achieving ultimate renaming control in complex scenarios. When facing a folder with a mix of images, videos, and documents, you can create one rule specifically for all .jpg files, another rule only for files with "report" in their names, and a third rule for all remaining files.

The scope feature works perfectly with the file filtering function, allowing you to first use filters to quickly locate specific file groups, and then set separate rules for these filtered files, achieving multi-level, multi-condition precise file processing.

A powerful file information center and editor. Select any file to instantly view its rich metadata with real-time editing support. Supports professional metadata extraction and management for multiple file types, allowing direct copying of metadata to renaming rules for more precise file organization.

Comprehensive support for MP3, FLAC, M4A, OGG, and more. Automatically extracts full ID3 tags (Title, Artist, Album, Composer, Genre, etc). Built-in editor allows direct modification and instant saving.

智能提取 Exif 元数据，涵盖相机型号、镜头参数、ISO、光圈、快门速度、拍摄时间及 GPS 地理位置信息。为摄影师提供精确的参数参考，并支持将拍摄时间用于文件名中。

Intelligently extracts Exif metadata including Camera Model, Lens, ISO, Aperture, Shutter Speed, Date Taken, and GPS. Provides precise data for photographers and supports using capture time in filenames.

Displays full technical specs: Codec, Resolution, Bitrate, Frame Rate. Deep structural analysis shows track counts, subtitle languages, and precise duration, giving you full control over video details.
