Whether you're a photographer, designer, or everyday user dealing with hundreds of files, this app handles it all with ease. With 13 versatile renaming rules, unlimited combinations, drag-and-drop reordering, and real-time previews, plus extensive metadata support and the innovative "rule scope" feature, Lightning Renamer transforms complex file organization into a simple task.

13 Professional Renaming Rules

替换

Replace

01/13

支持精确替换和模糊替换两种模式。精确替换要求完全匹配才进行替换；模糊替换则不区分大小写，能灵活匹配不同书写形式。可选择替换首个匹配项或全部匹配项，还支持反向功能，将替换内容变回原始内容。提供大小写匹配选项，确保替换结果的大小写与原始文本保持一致。

Supports both exact and fuzzy replacement modes. Exact replacement requires complete matching; fuzzy replacement is case-insensitive and can flexibly match different writing forms. You can choose to replace the first match or all matches, and it supports reverse functionality to change replacement content back to original. Provides case matching options to ensure the case of replacement results is consistent with the original text.

手术刀般精准的"规则作用域"功能

Surgical Precision with "Rule Scope"

如同外科手术刀般精准，规则作用域让您能够将每条规则精确定位到指定文件，在复杂场景下实现极致的重命名控制。当您面对一个混杂着图片、视频和文档的文件夹时，您可以创建一条规则专门作用于所有.jpg文件，另一条规则只作用于文件名包含"报告"的文档，而第三条规则则应用到其余所有文件上。

Like a surgeon's scalpel, Rule Scope enables you to precisely target each rule to specific files, achieving ultimate renaming control in complex scenarios. When facing a folder with a mix of images, videos, and documents, you can create one rule specifically for all .jpg files, another rule only for files with "report" in their names, and a third rule for all remaining files.

作用域功能与文件筛选功能完美结合，允许您先用筛选器快速定位特定文件群组（如某个时间段创建的文件、特定大小范围的文件或特定扩展名的文件），然后为这些筛选后的文件单独设置规则，实现多层次、多条件的精准文件处理。这种手术刀般的精细控制力，让您在最复杂的场景下也能游刃有余。

The scope feature works perfectly with the file filtering function, allowing you to first use filters to quickly locate specific file groups (such as files created within a specific timeframe, files of a certain size range, or files with specific extensions), and then set separate rules for these filtered files, achieving multi-level, multi-condition precise file processing. This surgical precision control lets you handle even the most complex scenarios with ease.

智能筛选系统

Intelligent Filtering System

多维度精确定位，海量文件从容管理。支持9大筛选类型的自由组合，通过与/或逻辑关系和分组功能构建复杂筛选条件，配合快速文本搜索和预设筛选方案，无论文件数量多少，都能快速找到目标文件。

Multi-dimensional precision targeting for effortless management of massive file collections. Supports flexible combination of 9 major filter types through AND/OR logic relationships and grouping functions. Combined with quick text search and preset filter schemes, quickly locate target files regardless of quantity.

基础筛选

Basic Filtering

提供9大核心筛选维度：文件名、新文件名、文件状态、文件类型、文件大小、修改日期、创建日期、参与状态和名称长度。每个维度都提供专门的比较运算符和匹配选项，如文本包含匹配、日期的范围选择、大小的区间设定等。内置快速文本搜索框，实时过滤文件列表；预设条件筛选提供常用筛选方案一键应用，大幅提升工作效率。

Offers 9 core filtering dimensions: File Name, New File Name, File Status, Kind, File Size, Modification Date, Creation Date, Participation Status, and Name Length. Each dimension provides specialized comparison operators and matching options, such as text contains matching, date range selection, and size interval settings. Built-in quick text search box for real-time file list filtering; preset condition filters provide one-click application of common filter schemes, significantly boosting work efficiency.

高级筛选

Advanced Filtering

突破单一条件限制，构建专业级复杂筛选逻辑。支持多层级的条件分组，每个分组可独立设置与（AND）或或（OR）逻辑关系；分组内可嵌套子分组，实现诸如"（文件大小大于10MB 且 格式为JPG）或（创建于最近7天 且 名称包含'项目'）"这样的多维度组合条件。提供可视化的条件树状结构，清晰展示筛选逻辑；支持对已保存的高级筛选条件进行再编辑和优化，让复杂场景下的文件定位变得简单直观。

Break through single-condition limitations to build professional-grade complex filtering logic. Supports multi-level hierarchical condition grouping, with each group independently configurable for AND or OR logic relationships; groups can contain nested subgroups, enabling multi-dimensional combinations like "(file size > 10MB AND format is JPG) OR (created within last 7 days AND name contains 'project')". Provides a visual tree structure for conditions, clearly displaying filter logic; supports re-editing and optimization of saved advanced filter conditions, making file targeting in complex scenarios simple and intuitive.
